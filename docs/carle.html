CARLE



    A table/heatmap showing grids updated per second in CARLE. With GPU execution and vectorization to a batch size of 128 reaches a maximum of more than 116,000 updates per second of a 64 by 64 grid. (Davis 2021)

  • CARLE (Cellular Automata Reinforcement Learning Environment) is a Life-like CA engine designed for human-machine collaborative creativity.

  • No reward in the env itself, but includes wrappers for curiosity-based rewards, mobility rewards, etc.

  • Can simulate any of 2^18 (262,144) Life-like CA rules, fast execution with vectorization and GPU acceleration, reads RLE files
